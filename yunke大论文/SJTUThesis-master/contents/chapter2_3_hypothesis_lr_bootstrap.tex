
\section{假设检验理论与似然比检验}

在计量经济学分析与时间序列建模中，对模型参数稳定性的统计推断具有基础性意义。无论是检验宏观经济系统的结构是否发生变化，还是识别金融变量之间动态关系是否出现断裂，均需要建立在明确的假设检验框架之上。本节从经典统计推断中的假设检验原理出发，进一步引入似然比检验的基本理论，并讨论其在向量自回归模型已知断点检验中的应用。随后，针对高维条件下经典渐近理论失效的问题，进一步介绍基于 Bootstrap 的非渐近重抽样推断思路。

\subsection{假设检验基本原理及核心评价指标}

假设检验的基本任务，是利用样本所提供的有限信息，对总体分布或参数所满足的某一命题作出统计决策。设概率空间为 $(\Omega,\mathcal{F},P)$，给定样本
\begin{equation}
X=(X_1,X_2,\dots,X_n),
\end{equation}
其联合密度函数或概率质量函数记为 $f(x;\theta)$，其中参数 $\theta$ 属于参数空间 $\Theta$。在统计推断中，通常将参数空间划分为两个互不相交的子集 $\Theta_0$ 与 $\Theta_1$，满足
\begin{equation}
\Theta_0\cup\Theta_1=\Theta, \qquad \Theta_0\cap\Theta_1=\emptyset.
\end{equation}
据此，原假设与备择假设分别写为
\begin{equation}
H_0: \theta\in\Theta_0, \qquad H_1: \theta\in\Theta_1.
\end{equation}

在给定假设体系后，需要构造一个基于样本的检验统计量 $T(X)$，并据此确定拒绝域 $R$。若实际样本对应的统计量落入拒绝域，则拒绝原假设；否则，不拒绝原假设。需要注意的是，“不拒绝”并不意味着原假设必然为真，而仅表示在当前样本信息下尚无充分证据反对原假设。

由于样本具有随机性，假设检验不可避免地会产生两类错误。第一类错误是指当原假设成立时错误地拒绝原假设，其概率记为
\begin{equation}
\alpha = \sup_{\theta\in\Theta_0} P(X\in R\mid \theta),
\end{equation}
通常称为检验的显著性水平或规模（size）。第二类错误是指当备择假设成立时未能拒绝原假设，其概率记为
\begin{equation}
\beta = P(X\notin R\mid \theta\in\Theta_1).
\end{equation}
与之对应，统计功效（power）定义为在备择假设成立时正确拒绝原假设的概率，即
\begin{equation}
1-\beta = P(X\in R\mid \theta\in\Theta_1).
\end{equation}

在有限样本条件下，第一类错误与第二类错误之间通常存在权衡关系。若为了降低第一类错误而过度收缩拒绝域，则会提高第二类错误，进而削弱检验功效。因此，经典假设检验的基本目标是在给定显著性水平 $\alpha$ 的前提下，尽可能提高检验功效。

这一思想在奈曼—皮尔逊引理（Neyman--Pearson Lemma）中得到了严格刻画。对于简单假设情形，即
\begin{equation}
H_0:\theta=\theta_0, \qquad H_1:\theta=\theta_1,
\end{equation}
奈曼—皮尔逊引理指出，在所有满足第一类错误概率不超过 $\alpha$ 的检验中，功效最大的最优检验必然具有似然比形式，其拒绝域可写为
\begin{equation}
R^*=\left\{x:\frac{L(\theta_1;x)}{L(\theta_0;x)}\geq k_{\alpha}\right\},
\end{equation}
其中，$L(\theta;x)$ 表示样本在参数 $\theta$ 下的联合似然函数，常数 $k_{\alpha}$ 由显著性水平约束唯一确定。该结论说明，似然比在提取样本关于假设差异的信息方面具有最优性，这也构成了广义似然比检验理论的基础。

\subsection{似然比检验理论及其高维局限性}

奈曼—皮尔逊引理主要适用于简单假设，而在大多数实际计量分析中，参数空间往往包含多个未知参数，因此检验问题通常表现为复合假设检验。为将似然比思想推广到此类情形，统计学中引入了广义似然比检验（Generalized Likelihood Ratio Test, GLRT）。其基本思想是，分别在受约束参数空间与不受约束参数空间中对似然函数进行最大化，并通过比较两者的最大似然值，评估原假设所施加约束对模型拟合优度的影响。

设样本似然函数为 $L(\theta;x)$，则广义似然比统计量定义为
\begin{equation}
\Lambda(x)=\frac{\sup_{\theta\in\Theta_0}L(\theta;x)}{\sup_{\theta\in\Theta}L(\theta;x)}.
\end{equation}
由于 $\Theta_0\subseteq\Theta$，故有 $0\leq \Lambda(x)\leq 1$。当 $\Lambda(x)$ 接近于 $1$ 时，表明原假设约束下的模型拟合与全模型差异较小，样本数据不足以反驳原假设；反之，当 $\Lambda(x)$ 接近于 $0$ 时，则说明原假设显著损害了模型对数据的解释能力，应倾向于拒绝原假设。

为了便于推断，通常对 $\Lambda(x)$ 进行对数变换，并构造统计量
\begin{equation}
G=-2\ln\Lambda(x)=2\left[\ln L(\hat{\theta}_{\mathrm{MLE}};x)-\ln L(\hat{\theta}_0;x)\right],
\end{equation}
其中，$\hat{\theta}_{\mathrm{MLE}}$ 为无约束极大似然估计，$\hat{\theta}_0$ 为原假设约束下的极大似然估计。根据威尔克斯定理（Wilks' theorem），在样本量趋于无穷且满足一系列正则性条件的低维环境下，统计量 $G$ 的极限分布为卡方分布：
\begin{equation}
G \xrightarrow{d} \chi_r^2,
\end{equation}
其中，$r$ 为受约束模型相对于全模型所施加的独立约束个数。

在时间序列分析中，似然比检验常被用于结构变化识别。以向量自回归模型的已知断点检验为例，考虑一个包含 $N$ 个内生变量、滞后阶数为 $p$ 的 VAR($p$) 模型：
\begin{equation}
Y_t=c+\Phi_1Y_{t-1}+\cdots+\Phi_pY_{t-p}+\epsilon_t,
\end{equation}
其中，误差项满足多元正态分布，协方差矩阵为 $\Sigma$。若已知样本中的某一时间点 $\tau$ 可能对应结构变化，则原假设 $H_0$ 表示整个样本期内参数保持稳定；备择假设 $H_1$ 表示系统在时间点 $\tau$ 发生结构性断裂，断点前后分别对应不同的参数矩阵。

在多元正态假设下，忽略常数项后，对数似然函数的核心部分可表示为残差协方差矩阵行列式的函数。对于长度为 $T$ 的样本，其极大对数似然可写为
\begin{equation}
\ln L \propto -\frac{T}{2}\ln|\hat{\Sigma}|.
\end{equation}
因此，在原假设下，利用全样本拟合受约束模型可得到残差协方差矩阵 $\hat{\Sigma}_R$；在备择假设下，对断点前后两个子样本分别拟合无约束模型，可得到残差协方差矩阵 $\hat{\Sigma}_1$ 与 $\hat{\Sigma}_2$。由此可构造已知断点下的对数似然比统计量：
\begin{equation}
LR = T_{\mathrm{eff}}\ln|\hat{\Sigma}_R| - T_1\ln|\hat{\Sigma}_1| - T_2\ln|\hat{\Sigma}_2|,
\end{equation}
其中，$T_{\mathrm{eff}}$ 为有效样本长度，$T_1$ 与 $T_2$ 分别表示断点前后两个子样本的长度。

在低维设定下，若 $N$ 与 $p$ 均较小，且样本规模足够大，则该统计量可近似服从卡方分布，其自由度等于断裂前后允许变化的参数个数。对于允许全部系数发生变化的 VAR($p$) 模型，约束数一般为
\begin{equation}
k = N(Np+1)=N^2p+N.
\end{equation}
因此，在经典低维框架中，研究者可以依据 $\chi_k^2$ 分布给出临界值或 $p$ 值，并据此完成结构变化检验。

然而，随着变量维度的增加，似然比检验的经典渐近理论会面临显著限制。问题的根源在于 VAR 模型参数维度随 $N^2p$ 速度增长。当变量维度较高而样本长度有限时，参数数量与样本量之比不再趋近于零，从而使威尔克斯定理所依赖的低维渐近条件不再成立。在这种情况下，对数似然函数的泰勒展开无法保证高阶余项可忽略，样本协方差矩阵的经验谱分布也会发生显著偏离，进而导致似然比统计量的均值和方差均偏离标准卡方分布所对应的理论值。

高维条件下，非约束模型往往能够更充分地拟合样本中的随机噪声，这会使似然比统计量出现系统性放大。若仍然沿用标准卡方分布表进行推断，则容易导致第一类错误率显著高于名义显著性水平。更进一步地，当参数维度与样本长度接近甚至超过样本长度时，非约束模型的残差自由度趋于耗尽，协方差矩阵可能变为奇异矩阵，从而使对数行列式无法计算，似然比统计量在代数意义上失去定义。因此，在高维 VAR 模型中，传统依赖渐近卡方分布的似然比检验往往难以维持可靠的有限样本推断性质。

\subsection{Bootstrap 自助法重抽样推断理论}

针对高维环境下似然比检验经典渐近理论失效的问题，Bootstrap 提供了一种重要的非渐近推断方法。Bootstrap 的基本思想是，不再完全依赖样本量趋于无穷时的解析极限分布，而是利用样本自身所包含的信息，通过重复重抽样来近似统计量在有限样本条件下的真实分布。其理论基础在于，当样本量足够大时，经验分布函数能够较好逼近总体分布，从而使基于样本构造的重抽样分布成为总体抽样分布的近似替代。

对于独立同分布样本，直接从原始样本进行重抽样通常即可实施 Bootstrap。然而，在时间序列模型中，观测值之间存在明显的动态相关结构，若直接对原始序列进行独立重抽样，将会破坏原有的时间依赖关系。基于这一原因，在 VAR 模型的结构变化检验中，更常采用残差 Bootstrap（residual bootstrap）方法。该方法将时间序列的生成机制分解为两个部分：一是由参数矩阵决定的系统性动态骨架，二是由残差所代表的随机冲击部分。通过在保持动态骨架不变的前提下，对残差进行随机重抽样并重新生成伪时间序列，便可以在原假设条件下构造出符合模型结构的 Bootstrap 样本。

在已知断点的 VAR 结构变化检验中，残差 Bootstrap 的基本步骤可以概括如下。首先，在原假设 $H_0$ 成立的条件下，利用全样本拟合受约束的 VAR($p$) 模型，得到参数估计值以及残差序列。其次，为避免有限样本下残差均值偏离零而影响重抽样结果，需要对残差进行中心化处理。记原始拟合残差为 $\hat{\epsilon}_t$，则中心化残差可定义为
\begin{equation}
\tilde{\epsilon}_t = \hat{\epsilon}_t - \bar{\hat{\epsilon}},
\end{equation}
其中，$\bar{\hat{\epsilon}}$ 表示残差样本均值向量。

随后，从中心化残差集合 $\{\tilde{\epsilon}_1,\tilde{\epsilon}_2,\dots,\tilde{\epsilon}_T\}$ 中进行有放回抽样，生成伪残差序列 $\epsilon_t^*$。为了保持原始 VAR 模型的动态结构，需保留初始若干期观测值作为递归起点，并利用原假设下估计得到的参数进行向前迭代，从而构造 Bootstrap 伪时间序列：
\begin{equation}
Y_t^* = \hat{c} + \hat{\Phi}_1^{(0)}Y_{t-1}^* + \cdots + \hat{\Phi}_p^{(0)}Y_{t-p}^* + \epsilon_t^*.
\end{equation}

由此生成的伪序列在构造逻辑上完全服从原假设，即系统参数在整个样本期内保持不变。接着，在每一条 Bootstrap 伪序列上重复执行与原始样本完全相同的检验流程：分别拟合受约束模型与非约束模型，计算对应的残差协方差矩阵，并据此得到一组 Bootstrap 似然比统计量，记为
\begin{equation}
LR_1^*,LR_2^*,\dots,LR_B^*.
\end{equation}
当重抽样次数为 $B$ 时，这些伪统计量就构成了原假设下似然比统计量的经验分布近似。

在获得经验分布后，可将原始样本对应的观测统计量 $LR_{\mathrm{obs}}$ 与其进行比较，从而计算经验 $p$ 值。其定义为
\begin{equation}
p\text{-value}=\frac{1}{B}\sum_{b=1}^{B}\mathbb{I}(LR_b^*\geq LR_{\mathrm{obs}}),
\end{equation}
其中，$\mathbb{I}(\cdot)$ 为指示函数。若该经验 $p$ 值小于预先设定的显著性水平 $\alpha$，则拒绝原假设，认为系统在给定断点处存在结构性变化；否则，不拒绝原假设。

从推断思想上看，Bootstrap 的优势在于它不依赖于标准卡方分布等解析渐近结果，而是直接利用样本所反映的有限样本结构构造统计量的经验分布。因此，在高维 VAR 模型中，当经典似然比检验的渐近分布近似严重失效时，Bootstrap 仍然能够为检验统计量提供更加可靠的临界值与 $p$ 值估计。尤其是在参数维度较高、样本量有限以及协方差矩阵谱结构复杂的情况下，Bootstrap 为结构变化检验提供了一种具有较强适应性的非渐近推断路径。

综上所述，经典假设检验理论为统计决策提供了基本框架，似然比检验则进一步将模型拟合优度比较引入复合假设情形，并在低维环境下依靠威尔克斯定理建立了明确的渐近分布基础。然而，在高维向量自回归模型中，参数维度膨胀会破坏这些经典结果的适用条件，使似然比检验面临明显的有限样本失真问题。Bootstrap 重抽样方法通过对原假设下数据生成机制的经验模拟，有效缓解了这一困难，从而成为高维结构变化推断中不可或缺的补充工具。
