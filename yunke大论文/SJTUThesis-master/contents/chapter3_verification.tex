\section{检验有效性的蒙特卡洛验证设计}\label{sec:mc-design}

在多元时间序列的实证分析中，对 VAR 模型参数稳定性的检验是关键环节。随着变量维度 $N$ 的增大，系数矩阵中的待估参数数量以 $O(N^2 p)$ 的速率增长，导致分段估计时有效自由度被快速消耗，似然比统计量在有限样本下的真实分布偏离渐近理论分布，进而引发检验水平失真。此外，在低秩约束或 $L_1$ 惩罚约束下，检验统计量不具备已知的解析渐近分布，数值模拟成为评估其统计特性的必要手段。

本节阐述评估第~\ref{sec:method-construction} 节所构建的结构性变化检验方案有效性的蒙特卡洛验证设计。该设计涵盖低维基准模型的行为对照，以及基于 Bootstrap 重抽样的似然比检验在中维稀疏与高维低秩结构下的统计性能评估。整个验证框架在已知断点的检验体系内展开，通过控制单一变量，分离不同估计方法与重抽样算法对推断结果的具体影响。

\subsection{模型分类与实验设定}\label{subsec:model-setup}

为覆盖从传统计量经济学到高维统计学习的不同范式，蒙特卡洛验证方案设定了以下几类具有代表性的数据生成与参数估计模型。

第一类为\textbf{稠密结构 OLS 基准模型}。在该类模型中，系数矩阵 $\Phi$ 的所有元素均为自由参数，采用普通最小二乘法（OLS）直接估计，并通过残差 Bootstrap 似然比检验进行推断。该类模型设置多个维度水平，用于考察 Bootstrap 方法在不同参数规模下的水平控制效果。

第二类为\textbf{渐近检验对比基准}。该模型同样采用 OLS 估计稠密结构的系数矩阵，但推断机制改为依赖解析分布的经典渐近检验（Chow $F$ 检验），用于与 Bootstrap 方法进行对比，揭示渐近分布在有限样本下的失真程度。

第三类为\textbf{中维稀疏结构 Lasso 模型}。当变量维度增大时，系数矩阵中仅有少数元素为非零值。该类模型采用引入 $L_1$ 惩罚项的 Lasso 逐方程回归进行估计，配合残差 Bootstrap 似然比检验进行推断。

第四类为\textbf{高维低秩结构 SVD 模型}。该类模型假定系数矩阵具有低秩性质，采用先 OLS 估计、后截断奇异值分解（SVD）的两步法进行估计，保留前若干个最大奇异值以实现低秩逼近，同样配合残差 Bootstrap 似然比检验进行推断。

在所有实验中，结构断点固定于样本中点 $t^* = T/2$。将断点位置预先设定的原因在于，本方案的首要目标是验证检验统计量在不同约束下的分布特性与判别能力，而非断点搜索算法的效率。通过固定断点位置，可以排除遍历搜索引入的计算噪声，评估 Bootstrap 方法对非标准分布的经验校正效果。同时，样本总长度的设定需满足以下约束：分段估计时，每段的有效样本量应足以支撑对应估计方法的参数拟合。对于高维模型，虽然原始参数量可能超过子样本量，但正则化或低秩截断将有效参数量压缩至可接受的范围，从而避免因自由度过度消耗导致的第一类错误率系统性偏高。各类模型的具体参数设定将在第~\ref{chap:experiment} 章中详细说明。


\subsection{第一类错误率的蒙特卡洛评估}\label{subsec:type1}

第一类错误率（Size）的评估旨在测量当 $H_0$ 成立时检验程序错误拒绝的概率。该阶段的实验环境设定为：VAR 模型的系数矩阵在整个观测期内保持恒定，系统不存在结构性变化。

\subsubsection{无结构变化时间序列的生成}

生成满足 $H_0$ 的时间序列，首先需构造满足不同结构约束的系数矩阵 $\Phi$。

\paragraph{稠密矩阵.}
$\Phi$ 的每个元素独立采样自均值为零的正态分布，标准差根据维度 $N$ 自适应调整，以提高生成矩阵满足平稳性条件的概率。

\paragraph{稀疏矩阵.}
在从正态分布中采样生成初始矩阵后，施加稀疏掩码：以预设的稀疏度参数 $s$ 保留每个元素，以概率 $1 - s$ 将其置零，由此模拟经济或金融网络中仅存在少数核心传导路径的稀疏结构。

\paragraph{低秩矩阵.}
生成两个低维因子矩阵 $U \in \mathbb{R}^{N \times r}$ 和 $V \in \mathbb{R}^{Np \times r}$，其元素独立采样自正态分布，随后通过 $\Phi = U V^\top$ 构造系数矩阵。该方式保证 $\mathrm{rank}(\Phi) = r$。

对于上述三类矩阵，均需执行平稳性校验：将 $\Phi$ 嵌入 $Np \times Np$ 维伴随矩阵 $\mathbf{A}$（式~\eqref{eq:companion}），计算其所有特征值，验证 $\max_i |\lambda_i(\mathbf{A})| < 1$。不满足平稳性条件的矩阵将被丢弃并重新生成。

获得平稳系数矩阵后，按 VAR($p$) 模型生成时间序列。为消除初始值对平稳分布的影响，在正式样本前额外生成一段预热数据并丢弃，使进入检验程序的有效样本具备平稳遍历特性。

\subsubsection{评估流程}

在每一次蒙特卡洛实验中，对随机生成的平稳时间序列执行完整的 Bootstrap 似然比检验流程（算法~\ref{alg:bootstrap-test}），获得经验 $p$ 值 $\hat{p}$。若 $\hat{p} \leq \alpha$，则记录为一次错误拒绝。经过 $M$ 次独立重复后，实际经验拒绝率为
\begin{equation}\label{eq:type1-rate}
    \hat{\alpha} = \frac{1}{M} \sum_{m=1}^{M} \mathbf{1}\{\hat{p}_m \leq \alpha\}.
\end{equation}
若检验程序的水平控制良好，则 $\hat{\alpha}$ 应接近名义水平 $\alpha$。二者之差 $\hat{\alpha} - \alpha$ 即为尺寸失真（size distortion）的度量。

该评估流程的伪代码描述见算法~\ref{alg:type1-mc}。

\begin{algorithm}[htbp]
\caption{第一类错误率蒙特卡洛评估}\label{alg:type1-mc}
\KwIn{时间序列参数 $(T, N, p)$，断点 $t^*$，显著性水平 $\alpha$，Bootstrap 次数 $B$，蒙特卡洛重复次数 $M$，估计方法}
\KwOut{经验拒绝率 $\hat{\alpha}$}

$\mathrm{rejections} \leftarrow 0$\;
\For{$m = 1$ \KwTo $M$}{
    依据模型结构生成平稳系数矩阵 $\Phi$，通过伴随矩阵校验平稳性\;
    丢弃预热期，生成无结构变化的平稳序列 $Y \in \mathbb{R}^{T \times N}$\;

    \tcp{受限模型拟合（$H_0$）}
    以指定估计方法对全样本拟合 VAR($p$)，提取 $\hat{\Phi}^{(0)}$, $\hat{c}^{(0)}$, $\{\hat{\varepsilon}_t\}$\;
    计算受限对数似然值 $\ell_0$\;

    \tcp{非受限模型拟合（$H_1$）}
    以 $t^*$ 为断点分段拟合，计算非受限对数似然值 $\ell_1 = \ell_{\text{seg1}} + \ell_{\text{seg2}}$\;
    $\mathrm{LR}_{\text{obs}} \leftarrow 2(\ell_1 - \ell_0)$\;

    \tcp{残差中心化}
    $\bar{\varepsilon} \leftarrow \frac{1}{T_{\text{eff}}} \sum_t \hat{\varepsilon}_t$\;
    $\tilde{\varepsilon}_t \leftarrow \hat{\varepsilon}_t - \bar{\varepsilon}$\;

    \tcp{Bootstrap 循环}
    $\mathrm{exceed} \leftarrow 0$\;
    \For{$b = 1$ \KwTo $B$}{
        从 $\{\tilde{\varepsilon}_t\}$ 中有放回抽样生成伪残差 $\{\tilde{\varepsilon}_t^{*}\}$\;
        以 $Y_{1:p}$ 为初始值，利用 $\hat{\Phi}^{(0)}$, $\hat{c}^{(0)}$ 和 $\{\tilde{\varepsilon}_t^{*}\}$ 递推生成伪序列 $Y^*$\;
        对 $Y^*$ 执行全样本与分段拟合，计算 $\mathrm{LR}^{*b}$\;
        \tcp{拟合方法需与原始数据一致}
        \If{$\mathrm{LR}^{*b} \geq \mathrm{LR}_{\emph{obs}}$}{
            $\mathrm{exceed} \leftarrow \mathrm{exceed} + 1$\;
        }
    }
    $\hat{p}_m \leftarrow \mathrm{exceed} / B$\;
    \If{$\hat{p}_m \leq \alpha$}{
        $\mathrm{rejections} \leftarrow \mathrm{rejections} + 1$\;
    }
}
$\hat{\alpha} \leftarrow \mathrm{rejections} / M$\;
\Return{$\hat{\alpha}$}\;
\end{algorithm}


\subsection{统计功效的评估方案与效应量设定}\label{subsec:power}

当 Bootstrap 似然比检验通过了第一类错误率的有效性验证后，验证设计的重点转向检验功效（Power）的评估。统计功效度量当 $H_1$ 成立——即系数矩阵在断点处确实发生变化时——检验能够正确拒绝 $H_0$ 的概率。

\subsubsection{效应量的定义}

为在不同维度和结构类型之间建立可比的度量基准，将效应量定义为断点前后系数矩阵之差的 Frobenius 范数：
\begin{equation}\label{eq:effect-size}
    \delta = \|\Phi_2 - \Phi_1\|_F = \left(\sum_{i,j} (\Phi_{2,ij} - \Phi_{1,ij})^2\right)^{1/2}.
\end{equation}
Frobenius 范数等价于将矩阵向量化后的欧氏距离，聚合了所有参数元素变化的总量。无论结构变化是集中于少数元素还是分散于大量元素，只要 $\delta$ 相同，注入系统的结构变化总信号强度即相同。验证方案在覆盖从微弱信号到强信号的离散效应量网格上评估功效。

\subsubsection{变异系数矩阵的构造}\label{subsubsec:phi2-construction}

给定断点前的平稳系数矩阵 $\Phi_1$ 和目标效应量 $\delta$，断点后系数矩阵 $\Phi_2$ 的构造需同时满足两个条件：(1) $\|\Phi_2 - \Phi_1\|_F$ 尽可能接近 $\delta$；(2) $\Phi_2$ 对应的 VAR 系统满足平稳性条件。

具体构造过程如下。首先确定扰动方向：构造与 $\Phi_1$ 同维度的全 $1$ 矩阵 $\mathbf{J}$，经 Frobenius 范数归一化得到单位方向矩阵
\begin{equation}\label{eq:direction}
    D = \frac{\mathbf{J}}{\|\mathbf{J}\|_F}.
\end{equation}
初始候选矩阵为 $\Phi_2^{(0)} = \Phi_1 + \delta \cdot D$。

若候选矩阵不满足平稳性条件（即伴随矩阵存在模大于等于 $1$ 的特征值），则启动收缩机制：引入收缩因子 $\gamma \in (0,1)$，令
\begin{equation}\label{eq:shrinkage}
    \Phi_2^{(k)} = \Phi_1 + \delta \cdot \gamma^k \cdot D, \quad k = 1, 2, \ldots, K_{\max},
\end{equation}
逐步缩小扰动幅度，直至 $\Phi_2^{(k)}$ 满足平稳性约束。

由于收缩操作的存在，$\Phi_2$ 的实际 Frobenius 距离 $\|\Phi_2 - \Phi_1\|_F$ 可能小于名义效应量 $\delta$。实验中记录实际效应量，以确保功效分析基于真实的参数距离。这一自适应特性反映了 VAR 系统在维持平稳性的同时所能容纳的最大结构变化幅度。

\subsubsection{备择假设下时间序列的生成}

在备择假设下，时间序列在断点 $t^*$ 前后分别由 $(\Phi_1, c_1)$ 和 $(\Phi_2, c_2)$ 驱动。具体生成方式为：(1) 以 $\Phi_1$ 为参数生成前 $t^*$ 期数据（含预热期，预热数据事后丢弃）；(2) 以 $\Phi_2$ 为参数生成 $t^* + 1$ 至 $T$ 期数据，其中第二段序列的初始滞后向量直接取用第一段序列末尾的观测值。这一衔接方式保证了序列在时间轴上的连续性，避免了因断点处的数据截断而导致的初始残差异常。

在两段数据上计算似然函数时，有效样本量的口径保持一致：第一段有效样本量为 $T_1 = t^* - p$，第二段为 $T_2 = T - t^*$，总量 $T_1 + T_2 = T - p = T_{\text{eff}}$。

\subsubsection{功效评估流程}

在每个效应量 $\delta$ 下，执行 $M$ 次蒙特卡洛实验。每次实验生成一条含结构变化的时间序列，对其执行完整的 Bootstrap 似然比检验流程。检验功效定义为正确拒绝 $H_0$ 的经验比例：
\begin{equation}\label{eq:power}
    \widehat{\mathrm{Power}}(\delta) = \frac{1}{M} \sum_{m=1}^{M} \mathbf{1}\{\hat{p}_m \leq \alpha\}.
\end{equation}
连接各效应量节点上的功效值，可绘制功效函数曲线 $\widehat{\mathrm{Power}}(\delta)$。该曲线反映了检验程序在不同信号强度下的判别能力：曲线上升越快，表明检验对结构变化越敏感。

\subsubsection{不同模型的分段估计策略}

在功效评估的分段拟合中，不同模型采用与全样本拟合一致的估计策略：
\begin{itemize}
    \item \textbf{OLS 模型}：对前后两段子样本分别执行 OLS 回归，直接获得局部系数估计。
    \item \textbf{Lasso 模型}：对前后两段子样本分别执行逐方程 Lasso 回归。坐标下降法的最大迭代次数需设定为足够大的值，以确保非光滑目标函数的充分收敛。
    \item \textbf{SVD 模型}：对前后两段子样本分别先执行 OLS 估计，再对所得系数矩阵进行奇异值分解，保留前 $r$ 个奇异值并将其余截断为零，由此重构降秩后的系数矩阵。该两步法先获取含噪的初始估计，再通过 Eckart--Young 定理的最优低秩逼近实现降噪。
\end{itemize}

在 Bootstrap 伪序列的重新拟合中，上述估计方法需一致地重新执行，以捕获正则化估计器在重抽样空间中的变异性。

该评估流程的伪代码描述见算法~\ref{alg:power-mc}。

\begin{algorithm}[htbp]
\caption{统计功效蒙特卡洛评估}\label{alg:power-mc}
\KwIn{时间序列参数 $(T, N, p)$，断点 $t^*$，显著性水平 $\alpha$，Bootstrap 次数 $B$，蒙特卡洛重复次数 $M$，效应量网格 $\{\delta_1, \ldots, \delta_K\}$，估计方法}
\KwOut{功效函数 $\{\widehat{\mathrm{Power}}(\delta_k)\}_{k=1}^{K}$}

\For{$k = 1$ \KwTo $K$}{
    $\mathrm{rejections} \leftarrow 0$\;
    \For{$m = 1$ \KwTo $M$}{
        生成平稳系数矩阵 $\Phi_1$\;
        以 $\Phi_1$ 和目标效应量 $\delta_k$ 构造变异矩阵 $\Phi_2$\;
        \tcp{通过收缩机制确保 $\Phi_2$ 满足平稳性条件}
        记录实际效应量 $\|\Phi_2 - \Phi_1\|_F$\;

        \tcp{生成含结构变化的时间序列}
        以 $\Phi_1$ 生成前 $t^*$ 期数据（含预热期）\;
        以 $\Phi_2$ 生成 $t^*+1$ 至 $T$ 期数据，初始滞后取第一段末尾观测值\;
        拼接为完整序列 $Y \in \mathbb{R}^{T \times N}$\;

        \tcp{执行 Bootstrap 似然比检验（算法~\ref{alg:bootstrap-test}）}
        对 $Y$ 进行全样本约束拟合，提取 $\hat{\Phi}^{(0)}$, $\hat{c}^{(0)}$, $\{\hat{\varepsilon}_t\}$，计算 $\ell_0$\;
        以 $t^*$ 为断点分段非约束拟合，计算 $\ell_1$\;
        $\mathrm{LR}_{\text{obs}} \leftarrow 2(\ell_1 - \ell_0)$\;
        中心化残差：$\tilde{\varepsilon}_t \leftarrow \hat{\varepsilon}_t - \bar{\varepsilon}$\;

        $\mathrm{exceed} \leftarrow 0$\;
        \For{$b = 1$ \KwTo $B$}{
            从 $\{\tilde{\varepsilon}_t\}$ 中有放回抽样生成伪残差\;
            以 $\hat{\Phi}^{(0)}$, $\hat{c}^{(0)}$ 递推生成伪序列 $Y^*$\;
            对 $Y^*$ 执行全样本与分段拟合，计算 $\mathrm{LR}^{*b}$\;
            \If{$\mathrm{LR}^{*b} \geq \mathrm{LR}_{\emph{obs}}$}{
                $\mathrm{exceed} \leftarrow \mathrm{exceed} + 1$\;
            }
        }
        $\hat{p}_m \leftarrow \mathrm{exceed} / B$\;
        \If{$\hat{p}_m \leq \alpha$}{
            $\mathrm{rejections} \leftarrow \mathrm{rejections} + 1$\;
        }
    }
    $\widehat{\mathrm{Power}}(\delta_k) \leftarrow \mathrm{rejections} / M$\;
}
\Return{$\{\widehat{\mathrm{Power}}(\delta_k)\}_{k=1}^{K}$}\;
\end{algorithm}

通过上述设计，蒙特卡洛验证方案构建了一个不依赖解析分布的数值实验平台，能够在高维、正则化约束等非标准条件下定量评估 Bootstrap 似然比检验的水平控制能力和检验功效，为实证应用中判断检验工具的适用范围提供依据。
