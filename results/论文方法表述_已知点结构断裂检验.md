# 论文方法表述：已知点结构断裂检验

## 1. 研究问题

本文研究已知时间点下 VAR(p) 模型的结构断裂检验问题。设观测序列为 `Y_1, ..., Y_T`，给定断点 `t`，检验该点前后模型参数是否发生变化。

原假设与备择假设写为：

```text
H0: 断点 t 处不存在结构断裂，断点前后参数相同；
H1: 断点 t 处存在结构断裂，断点前后参数不同。
```

等价地，可写为：

```text
H0: Φ1 = Φ2
H1: Φ1 ≠ Φ2
```

其中 `Φ1` 表示断点前参数，`Φ2` 表示断点后参数。

---

## 2. 模型设定

在 `H0` 下，整条时间序列共用一套 VAR(p) 参数：

```text
Y_s = c + Φ X_s + ε_s,   s = p+1, ..., T
```

其中 `X_s = (Y_{s-1}', ..., Y_{s-p}')'` 为滞后向量。

在 `H1` 下，序列在已知点 `t` 处发生结构断裂，断点前后分别使用两套参数：

```text
Y_s = c1 + Φ1 X_s + ε_s,   s = p+1, ..., t
Y_s = c2 + Φ2 X_s + ε_s,   s = t+1, ..., T
```

这里采用单序列、双参数体制的条件似然口径：第二段从断点后第一个响应开始生效，但其回归所需的前 `p` 个滞后值允许借用断点前观测。因此，断点后的样本不是从 `Y_{t+1}` 独立重启，而是延续同一条时间序列的条件动态结构。

---

## 3. 样本量口径

在上述设定下：

- `H0` 的有效样本量为 `T-p`；
- `H1` 的第一段有效样本量为 `t-p`；
- `H1` 的第二段有效样本量为 `T-t`。

因此，`H1` 的总有效样本量为：

```text
(t-p) + (T-t) = T-p
```

这意味着 `H0` 与 `H1` 在比较时使用相同的有效样本量口径，从而避免了旧版“分段后两段各自损失前 `p` 个观测”导致的样本量不一致问题。

---

## 4. LR 统计量

令 `L0` 表示 `H0` 下的对数似然，`L1` 表示 `H1` 下的对数似然，则似然比统计量定义为：

```text
LR(t) = 2 [L1(t) - L0]
```

对于低维 OLS VAR，在高斯误差与常规正则条件下，该统计量可作为已知点结构断裂检验的核心统计量；在固定维度、固定阶数且断点不位于边界时，其渐近分布可与标准已知断点 Chow / LR 检验理论对应。

---

## 5. Bootstrap 推断

考虑到有限样本下渐近近似可能不稳定，本文采用残差 bootstrap 对 LR 统计量进行校正：

1. 在 `H0` 下拟合受限模型，得到参数估计与残差；
2. 对残差居中并有放回重抽样；
3. 在 `H0` 下递归生成 bootstrap 伪样本；
4. 对每个伪样本重复同样的结构断裂检验，得到 `LR*_1, ..., LR*_B`；
5. 以经验分布计算 p 值：

```text
p_value = P(LR* >= LR_obs)
```

若 `p_value <= alpha`，则拒绝 `H0`，认为已知点 `t` 处存在结构断裂。

---

## 6. 三类模型的统一解释

本文的三类模型分别为：

- `baseline_ols`：低维 OLS + 已知点结构断裂检验 + bootstrap LR
- `sparse_lasso`：稀疏高维 VAR + 已知点结构断裂检验 + bootstrap LR
- `lowrank_svd`：低秩高维 VAR + 已知点结构断裂检验 + bootstrap LR

三类模型在检验层面使用统一的 `H0/H1` 与样本量口径；差异仅在于参数估计器不同。
