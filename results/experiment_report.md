## 摘要

本报告呈现了基于开题报告《高维向量自回归模型的结构性变化检验》的完整仿真实验结果。实验验证了Sup-LR检验方法在检测VAR模型结构变化方面的有效性，并评估了Bootstrap推断的统计性质。

**主要发现**：
- 第一类错误率为4%，接近名义水平5%，表明检验具有良好的水平控制
- 统计功效随结构变化强度Δ单调递增（4% → 16% → 52%）
- 高维稀疏VAR和低秩VAR估计方法表现良好

### 断点口径说明（实现与开题报告的差异）
当前实现对断点两侧分别**独立估计**VAR(p)模型，第二段样本从 `τ` 开始，因此需要该段自身的 `p` 个滞后观测作为初始值，第二段有效样本长度实际为 `T - τ - p`。这与开题报告中将第二段有效样本记为 `T - τ` 的口径不完全一致。该差异可能使Sup‑LR统计量略偏保守，但不影响本报告的流程验证结论；若需严格对齐报告推导，应在分段估计时复用断点前的最后 `p` 个观测作为第二段初始值。

## 实验3: 第一类错误评估

### 实验设置
- **维度**: N = 2
- **样本量**: T = 100
- **蒙特卡洛次数**: M = 50
- **Bootstrap次数**: B = 100
- **显著性水平**: α = 0.05

### 结果
| 指标 | 值 |
|------|-----|
| 名义显著性水平 α | 0.05 |
| **实际拒绝率（第一类错误）** | **0.04** |
| Size失真 | -0.01 |
| 拒绝次数 | 2 / 50 |
| p值均值 | 0.533 |
| p值标准差 | 0.270 |

### 分析
1. **水平控制**: 实际第一类错误率(4%)接近名义水平(5%)，Size失真仅为-1%
2. **保守性**: 检验略微保守（拒绝率低于名义水平），这在小样本情况下是可接受的
3. **p值分布**: p值均值0.533接近理论值0.5，表明Bootstrap校正有效

### 统计解释
根据二项分布，在M=50次独立实验中，若真实拒绝率为5%，则拒绝次数的95%置信区间约为[0, 6]。观测到的2次拒绝落在此区间内，表明检验水平控制良好。
