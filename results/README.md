# Results 目录说明

本目录用于存放所有仿真实验的结果文件。

## 目录结构

```
results/
├── README.md                           # 本文件
├── experiment_report.md                # 全面实验报告（Sup-LR检验）
├── experiment_results.json             # 实验结果数据（JSON格式）
└── 仿真结果分析报告_2026-02-05.md      # 针对特定点LR检验的分析报告
```

## 两种检验方法的对比

| 特性 | Sup-LR检验 | 针对特定点LR检验 |
|------|-----------|-----------------|
| 报告文件 | experiment_report.md | 仿真结果分析报告_2026-02-05.md |
| 计算复杂度 | O(T) | O(1) |
| 适用场景 | 未知变点位置 | 已知/怀疑特定时间点 |
| 统计功效 | 较低（搜索成本） | 较高（针对性强） |
| 多重检验 | 需要校正 | 无 |

## 命名规范

建议未来的实验报告遵循以下命名规范：

```
仿真结果分析报告_[日期]_[实验类型].md
```

例如：
- `仿真结果分析报告_2026-02-05_特定点检验.md`
- `仿真结果分析报告_2026-02-06_高维稀疏VAR.md`
- `仿真结果分析报告_2026-02-07_功效曲线分析.md`
