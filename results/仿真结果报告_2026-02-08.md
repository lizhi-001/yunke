## 摘要

本报告基于仿真方案（`simulation_plan.md`）的完整实验设计，系统评估了VAR模型结构性变化检验方法的统计性能。实验涵盖两套完整检验流程：

1. **正常VAR模型检验**（Sup-LR + Bootstrap）：适用于低维场景
2. **高维稀疏/低秩VAR模型检验**（Lasso/核范数正则化）：适用于高维场景

同时对比了**Sup-LR检验**（遍历所有可能断点）与**针对特定时间点的LR检验**两种方法。

**主要发现**：
- Sup-LR检验的第一类错误率为4%，接近名义水平5%，Size控制良好
- 统计功效随结构变化强度Δ单调递增（4% → 16% → 52%），符合理论预期
- 针对特定点的LR检验第一类错误率为6.67%，略高于名义水平
- 高维稀疏VAR（Lasso）和低秩VAR（核范数正则化）估计方法均表现良好
- 断点定位精度随结构变化强度增大而提升

### 1.2 断点口径说明

当前实现对断点两侧分别**独立估计**VAR(p)模型，第二段样本从 `τ` 开始，需要该段自身的 `p` 个滞后观测作为初始值，第二段有效样本长度实际为 `T - τ - p`。这与开题报告中将第二段有效样本记为 `T - τ` 的口径不完全一致。该差异可能使统计量略偏保守，但不影响流程验证结论。

## 二、正常VAR模型检验结果

### 实验1: 基础VAR模型检验（H0为真）

**目的**: 验证在无结构变化时，Sup-LR检验能否正确接受原假设。

**实验设置**:
- 维度 N = 3，样本量 T = 200，滞后阶数 p = 1，Bootstrap次数 B = 200

**系数矩阵**:
```
Φ = [[ 0.149  -0.041   0.194]
     [ 0.457  -0.070  -0.070]
     [ 0.474   0.230  -0.141]]
```
平稳性检验: ✅ 通过

**检验结果**:

| 指标 | 值 |
|------|-----|
| Sup-LR统计量 | 46.50 |
| 估计断点位置 | 40 |
| Bootstrap p值 | 0.130 |
| 临界值 (α=0.10) | 48.25 |
| 临界值 (α=0.05) | 50.92 |
| 临界值 (α=0.01) | 54.89 |
| **决策** | **接受H0：不存在结构性变化** |

**分析**: 在无结构变化的数据上，检验正确地接受了原假设（p值=0.130 > 0.05）。Sup-LR统计量46.50低于5%临界值50.92，表明检验在H0为真时能够正确控制第一类错误。

---

### 实验2: 含结构断点的VAR模型检验（H1为真）

**目的**: 验证在存在结构变化时，Sup-LR检验能否正确拒绝原假设并定位断点。

**实验设置**:
- 维度 N = 3，样本量 T = 200，滞后阶数 p = 1，真实断点 τ = 100，B = 200

**系数变化**:
- 断点前 Φ₁: 稳定系数矩阵（scale=0.2）
- 断点后 Φ₂: 显著变化的系数矩阵
- 系数变化量（Frobenius范数）: **1.873**

**检验结果**:

| 指标 | 值 |
|------|-----|
| Sup-LR统计量 | 177.41 |
| 估计断点位置 | 93 |
| 真实断点位置 | 100 |
| 断点估计误差 | 7 |
| Bootstrap p值 | 0.000 |
| **决策** | **拒绝H0：存在结构性变化** |

**分析**:
1. **检验功效**: 在存在显著结构变化（‖Φ₂-Φ₁‖_F = 1.873）时，检验成功拒绝了原假设（p值=0.000）
2. **断点定位**: 估计断点(93)与真实断点(100)相差7个时点，定位精度较好
3. **统计量强度**: Sup-LR=177.41远超所有临界值，表明结构变化信号强烈

---

### 实验3: Sup-LR检验第一类错误评估

**目的**: 通过蒙特卡洛仿真评估Sup-LR检验的Size控制。

**实验设置**:
- 维度 N = 2，样本量 T = 100，M = 50，B = 100，α = 0.05

**结果**:

| 指标 | 值 |
|------|-----|
| 名义显著性水平 α | 0.05 |
| **实际拒绝率（第一类错误）** | **0.04** |
| Size失真 | -0.01 |
| 拒绝次数 | 2 / 50 |
| 有效迭代次数 | 50 / 50 |
| p值均值 | 0.533 |
| p值标准差 | 0.270 |

**分析**:
1. **水平控制**: 实际第一类错误率(4%)接近名义水平(5%)，Size失真仅为-1%
2. **保守性**: 检验略微保守（拒绝率低于名义水平），在小样本情况下可接受
3. **p值分布**: p值均值0.533接近理论值0.5，表明Bootstrap校正有效
4. **统计解释**: 根据二项分布，M=50次独立实验中若真实拒绝率为5%，拒绝次数的95%置信区间约为[0, 6]。观测到的2次拒绝落在此区间内

### 实验4: Sup-LR检验统计功效评估

**目的**: 评估Sup-LR检验在不同结构变化强度下的功效。

**实验设置**:
- 维度 N = 2，样本量 T = 100，M = 50，B = 100，真实断点 τ = 50

**功效曲线**:

| 结构变化强度 Δ | 系数变化量 | 功效 (Power) | 断点估计偏差 | 断点估计RMSE |
|----------------|------------|--------------|--------------|--------------|
| 0.1 | 0.20 | **4%** | 1.22 | 19.69 |
| 0.2 | 0.40 | **16%** | 1.34 | 18.22 |
| 0.3 | 0.60 | **52%** | -1.08 | 12.33 |

**功效曲线图示**:
```
Power
  |
60%|                              *
   |                            /
50%|                          /
   |                        /
40%|                      /
   |                    /
30%|                  /
   |                /
20%|              *
   |            /
10%|          /
   |        /
 5%|------*--------------------- α=0.05
   |
 0%+----+----+----+----+----+----
      0.0  0.1  0.2  0.3  0.4  Δ
```

**分析**:
1. **单调性**: 功效随Δ单调递增，符合理论预期
2. **敏感性**: 当Δ=0.1时功效接近α水平（4%），此时结构变化难以检测
3. **有效性**: 当Δ=0.3时功效达到52%，检验具有实际应用价值
4. **断点定位**: 随着Δ增大，断点估计RMSE从19.69降至12.33，定位精度提升
5. **偏差特性**: 断点估计偏差较小（|bias| < 1.5），无系统性偏移

## 三、针对特定时间点的LR检验结果

### Sup-LR检验 vs 特定点LR检验对比

| 特性 | Sup-LR检验 | 特定点LR检验 |
|------|-----------|-------------|

| 多重检验问题 | 需要Bootstrap校正 | 无多重检验问题 |

## 四、高维稀疏VAR模型结果

### 实验5: Lasso-VAR估计

**目的**: 验证Lasso估计在高维稀疏VAR模型中的参数恢复能力。

**实验设置**:
- 维度 N = 10，样本量 T = 300，滞后阶数 p = 1，目标稀疏度 20%

**结果**:

| 指标 | 值 |
|------|-----|
| 真实稀疏度 | 81% |
| 真实非零系数数量 | 19 / 100 |
| 估计稀疏度 | 62% |
| 估计非零系数数量 | 38 / 100 |
| 平均正则化参数 λ | 0.0309 |
| 估计误差（Frobenius范数） | 0.392 |
| 对数似然值 | -3135.05 |

**分析**:
1. **稀疏恢复**: Lasso估计成功识别了稀疏结构，但估计的非零系数数量(38)多于真实值(19)，存在一定的假阳性
2. **估计精度**: Frobenius范数误差0.392表明参数估计精度较好
3. **正则化选择**: 5折交叉验证选择的λ=0.0309实现了稀疏性与拟合精度的平衡

**目的**: 验证核范数正则化方法在低秩VAR模型中的秩恢复和参数估计能力。

```
真实系数矩阵奇异值:
σ₁ = 0.713, σ₂ = 0.520, σ₃ ≈ 0, σ₄ ≈ 0, ...
```

前两个奇异值显著非零，其余接近机器精度，确认真实秩为2。

**分析**:
1. **秩选择**: BIC准则正确选择了真实秩r=2，特征值比例法略有高估(r=3)
2. **估计精度**: Frobenius范数误差0.430与稀疏VAR相当
3. **方法有效性**: 截断SVD方法成功恢复了低秩结构

## 七、结论与建议

### 7.1 主要结论

1. **Sup-LR检验有效性**: Sup-LR + Bootstrap检验方法在检测VAR模型结构变化方面表现良好，第一类错误率(4%)接近名义水平(5%)
2. **功效特性**: 功效随结构变化强度Δ单调递增（4% → 16% → 52%），符合理论预期
3. **特定点检验**: 针对特定时间点的LR检验在已知变点位置时效率更高，但Size控制略逊于Sup-LR
4. **高维扩展**: Lasso-VAR和低秩VAR估计方法为高维场景提供了有效工具，参数恢复精度良好
5. **数值稳定性**: 代码改进后所有实验均无数值异常，Bootstrap校正有效
