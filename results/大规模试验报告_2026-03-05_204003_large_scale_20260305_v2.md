## 3. Baseline：渐近 p 值 vs Bootstrap p 值（随 B 变化）

**数据背景（本节口径）**：
- 本节基于 **H0（无断点）** 的同一条 baseline VAR 样本计算：`N=2, T=120, p=1, t=60`。
- 误差项设定：`u_t ~ N(0, 0.5I)`。
- 对固定 `LR_obs` 比较 `χ² p值(LR)` 与 `bootstrap LR p值`。

| B | χ² p值 (LR) | F渐近p值 | bootstrap LR p值 | bootstrap F p值 | |bootstrap LR - χ²| |
|---:|---:|---:|---:|---:|---:|
| 200  | 0.37299 | 0.35923 | 0.44000 | 0.40500 | 0.06701 |
| 500  | 0.37299 | 0.35923 | 0.42800 | 0.35200 | 0.05501 |
| 1000 | 0.37299 | 0.35923 | 0.42600 | 0.35700 | 0.05301 |

**解读**：
- `B` 增加后，`|bootstrap LR - χ²|` 整体下降（0.067 → 0.053），但仍有可见差距。
- 这说明在该有限样本口径下，bootstrap 与纯渐近卡方近似并非完全一致。

## 4. Monte Carlo：第一类错误与功效（随 M 变化）

**数据背景（本节口径）**：
- 指标定义：`第一类错误` = H0 为真时 `p ≤ α` 的比例；`功效` = H1 为真时 `p ≤ α` 的比例。
- 显著性水平统一为 `α=0.05`。
- `M` 表示在 H0 与 H1 下分别进行的 Monte Carlo 重复次数（每个 M 下各做 M 次），实际成功次数见 `M_effective_type1` 与 `M_effective_power`。
- Baseline 场景：`N=2, T=120, p=1, t=60`；H0 用单一 `Phi1` 生成；H1 用 `Phi2=ensure_stationary(Phi1+0.25*1)` 生成。
- 稀疏场景：`N=5, T=200, p=1, t=100`；`Phi1` 为稀疏（`sparsity=0.2`）；H1 用 `Phi2=ensure_stationary(Phi1+0.25*1)`。
- 低秩场景：`N=8, T=200, p=1, t=100`；`Phi1` 为低秩（`rank=2`）；H1 用 `Phi2=ensure_stationary(Phi1+0.20*1)`。
- 本次运行中，高维（稀疏/低秩）场景内部 bootstrap 次数为 `B_mc=50`。

### 4.1 Baseline 四种推断并行结果
| 模型 | M=50 (Type I / Power) | M=100 (Type I / Power) | M=200 (Type I / Power) |
|---|---|---|---|
| baseline_chow_asym_f | 0.04 / 0.72 | 0.08 / 0.73 | 0.035 / 0.765 |
| baseline_chow_asym_chi2 | 0.06 / 0.66 | 0.10 / 0.74 | 0.05 / 0.795 |
| baseline_chow_bootstrap_f | 0.04 / 0.64 | 0.08 / 0.70 | 0.035 / 0.705 |
| baseline_chow_bootstrap_lr | 0.04 / 0.60 | 0.09 / 0.67 | 0.035 / 0.715 |

### 4.2 高维场景结果
| 模型 | M=50 (Type I / Power) | M=100 (Type I / Power) | M=200 (Type I / Power) |
|---|---|---|---|
| sparse_lasso | 0.12 / 0.98 | 0.09 / 0.99 | 0.075 / 0.995 |
| lowrank_svd | 0.12 / 1.00 | 0.11 / 1.00 | 0.07 / 1.00 |

**解读**：
- Baseline 在 `M=200` 时，Type I Error 更接近目标 `0.05`（尤其 asym-χ²=0.05）。
- Baseline 功效在 `0.705~0.795` 区间，体现出中等断点强度下的可检测性。
- 稀疏/低秩场景功效很高，但 Type I Error 仍高于 0.05（尤其 `M=50/100`），后续可通过增大 `M`、调参与 bootstrap 次数继续校准。

## 5. 结论与下一阶段扩展
1. 当前修订框架（baseline 并行 F/LR + bootstrap，高维走 LR+bootstrap）在较大规模下可稳定运行并导出完整图表。
2. Baseline 的 bootstrap 与 χ² 近似存在有限样本差距，支持“并行报告 + 对比解释”的必要性。
3. 下一阶段按以下配置扩展：
   - `M-grid` 扩展到 `300, 500, 800`
   - 高维场景 `B_mc` 提升到 `100~200`
   - 对同一配置做多 seed 重复，报告均值与置信区间。
