### 4.1 针对特定点检验的优势

1. **计算效率高**
   - 无需遍历所有可能断点
   - 计算复杂度从O(T)降至O(1)

2. **适用场景明确**
   - 适合已知或怀疑特定时间点存在变化的情况
   - 例如：政策变化、市场事件等外生冲击

3. **统计推断清晰**
   - 直接针对特定假设进行检验
   - 避免了多重检验问题

### 4.2 与Sup-LR检验的对比

| 特性 | 针对特定点LR检验 | Sup-LR检验 |
|------|-----------------|-----------|
| 计算复杂度 | O(1) | O(T) |
| 适用场景 | 已知变点位置 | 未知变点位置 |

### 4.3 Bootstrap方法的有效性

**优点**:
- 有效处理了LR统计量的非标准分布
- 无需依赖渐近理论假设
- 适用于小样本和高维场景

**局限性**:
- 计算成本较高（需要B次重抽样）
- 对残差分布的假设敏感
- 在极小样本下可能不稳定

### 6.2 核心算法

**LR统计量计算**:
```
LR(t) = 2[lnL_U(t) - lnL_R]
其中:
- lnL_R: H0下的对数似然（全样本估计）
- lnL_U(t): H1下的对数似然（在t处分段估计）
```

**Bootstrap推断**:
```
1. 在H0下估计参数Φ̂, ĉ, 残差ε̂
2. 重抽样残差生成伪序列Y*
3. 计算伪序列的LR*(t)
4. 重复B次，构建经验分布
5. 计算p值 = P(LR* ≥ LR_observed)
```
